  • Bienvenido!

    SWC es una aplicación web que utiliza tecnología de la web semántica para mostrar información de entidades comerciales, tales como horarios, ubicación, métodos de pagos, página web, productos o servicios que ofrecen, entre otros. Cuenta con dos formas de encontrarlos, la primera es por tu ubicación actual, es decir la aplicación detecta donde te encuentras y te muestra los negocios a tu alrrededor. Y la segunda es estableciendo una dirección en especifico.

      SWC cuenta con otras características como: filtrar resultados, determinar a que distancia se encuentra el negocio, mostrar mapa, especificar si el negocio esta abierto o cerrado.

      Leer más

      La aplicación utiliza como base de conocimientos la ontología GoodRelations. La ontología es utilizada para describir con precisión empresas y lo que ofrecen.

  • Web Semántica

    La investigación de la Web Semántica se ha traducido en los últimos años con resultados significativos. Varias industrias han adoptado tecnologías de Web Semántica, mientras que la "Web actual" se encuentra en un punto de transformación crítico, puesto que la mayoría de los datos que se encuentran en la Web están siendo explotados a través de capas de la Web Semántica. Por ejemplo, el comercio electrónico, es una plataforma en la Web donde los compradores y vendedores se reúnen y crean un negocio, pero sólo tiene éxito cuando el comprador encuentra el producto que busca en el momento adecuado y se lleva a cabo una transacción comercial. En este caso, la Web Semántica proporciona transacciones de comercio electrónico y una plataforma elegante de espacio en el que los requerimientos de los clientes del producto y las prioridades se administran al mismo tiempo, junto con la cartera de productos. La transacción de Web Semántica se puede crear, compartir y reutilizar de acuerdo con las políticas de la organización de comercio electrónico. De esta forma, la Web Semántica ofrece al cliente un espacio inteligente para productos, servicios y transacciones de comercio electrónico. Por otro lado, una ontología es una conceptualización de un dominio en un formato humano comprensible, legible por máquina que consta de entidades, atributos, relaciones y axiomas. Se utiliza como una representación del conocimiento estándar para la Web Semántica.
